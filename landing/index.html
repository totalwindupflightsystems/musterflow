Four surfaces, one binary

🖥️

Web Dashboard

Browse connected APIs, search the community catalog, manage workflows. Local, self-hosted, zero telemetry.

⌨️

CLI

Every connected API becomes typed subcommands. musterflow gh issues list. musterflow stripe charges list. Bash/zsh/fish completion included.

🤖

MCP Endpoint

AI agents (Claude, GPT, Hermes) get every connected API as an MCP tool. No integration code. Just tools.

⚙️

Workflow Engine

Chain APIs with Starlark DSL. Webhook triggers. WASM transforms. "When Stripe charge succeeds → create GitHub issue."

The Context7 model — community-powered catalog

Anyone can add an API. Anyone can share a workflow. Anyone can publish a WASM transform. Quality scoring surfaces the good ones. The catalog is a public GitHub repo — no gatekeepers, no SaaS dependency, no registration required.

$ musterflow connect https://api.github.com — 42 CLI commands + MCP tools, instantly.

$ musterflow catalog push gh — share it with the community.

$ musterflow flow create → on stripe:charge.succeeded → gh issues create(title="Payment: $amount")

Install

# macOS / Linux
brew install musterflow

# Go
go install github.com/totalwindupflightsystems/musterflow/cmd/musterflow@latest

# Docker
docker run -p 9876:9876 -v ~/.musterflow:/root/.musterflow ghcr.io/totalwindupflightsystems/musterflow:latest

Requirements: Go 1.26+ if building from source. MIT licensed.
