⚡ MusterFlow

Turn any API into a CLI, an MCP tool, and a workflow.

Quick Start

1. Connect an API: musterflow connect https://api.github.com

2. Use the CLI: musterflow gh issues list

3. Point your MCP client to: http://localhost:9876/mcp

Connected APIs

No APIs connected. Run musterflow connect <url> to add one.

MCP Endpoint

URL: http://localhost:9876/mcp

Transport: stdio JSON-RPC / HTTP SSE

Connect any MCP-compatible client to use connected APIs as tools.
