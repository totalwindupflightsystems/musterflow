⚡ MusterFlow

Turn any API into a CLI, an MCP tool, and a workflow.

Quick Start

1. Connect an API: musterflow connect https://api.github.com

2. Use the CLI: musterflow gh issues list

3. Point your MCP client to: http://localhost:9876/mcp
